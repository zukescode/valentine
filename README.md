# valentine

Static Valentine "Yes/No" site.

## Run locally

Open `valentine/index.html` in a browser.

## Host on GitHub Pages

This repo includes a GitHub Actions workflow that deploys the `valentine/` folder to GitHub Pages.

1. Push to the `main` branch.
2. In your GitHub repo: **Settings → Pages → Build and deployment → Source** = **GitHub Actions**.
3. Wait for the "Deploy Valentine site to GitHub Pages" workflow to finish.

Your site will be available at the URL shown in the workflow output.
